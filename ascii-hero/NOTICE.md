The ASCII shader and 3D scene logic in `src/AsciiEffect.jsx` and `src/EffectScene.jsx`
is ported from [egorshest/webgl-ascii-hero](https://github.com/egorshest/webgl-ascii-hero)
(MIT License, see `LICENSE`), converted from TypeScript/Next.js to plain
JavaScript/Vite for embedding in a static site. The 3D model
(`../models/hero-model.glb`) is original, generated for this project.
